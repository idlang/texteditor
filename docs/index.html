Text Editor

A text selection range API written in pure JavaScript, for modern browsers.

Download Documentation

Start

You need to include at least the base CSS file called ui.min.css, a Font Awesome library and three JavaScript files:

  • text-editor.min.js → this is the core API for doing text area–based text selection manipulation.
  • ui.min.js → this is a module to create the GUI appearance as you can see in the text editor above.
  • html.min.js → this is a HTML Text Editor plugin for the user interface module, contains tools for editing HTML source.
<!DOCTYPE html>
<html dir="ltr">
  <head>
    <meta charset="utf-8">
    <title>Test Editor</title>
    <link href="font-awesome/font-awesome.min.css" rel="stylesheet">
    <link href="text-editor/ui/ui.min.css" rel="stylesheet">
  </head>
  <body>
    <p><textarea id="editor"></textarea></p>
    <script src="text-editor.min.js"></script>
    <script src="text-editor/ui/ui.min.js"></script>
    <script src="text-editor/ui/html/html.min.js"></script>
    <script>
    var editor = TE.ui.HTML(document.querySelector('#editor'));
    </script>
  </body>
</html>

Configure

TE.ui.HTML() contains some default options which you can override it through the second parameter:

var editor = TE.ui.HTML(document.querySelector('#editor'), {
    // options goes here…
});

Configuration

The available configuration data are:

  • Inherits TE.ui()
    • tab → string · set custom indent character.
    • dir → string · editor language direction (currently this option will only set a HTML · dir attribute to the text editor container).
    • resize → boolean · enable/disable text area resize handler.
    • keys → boolean · enable/disable keyboard shortcut feature.
    • tools → string · list of tools to be included to the tools bar, separated by a space.
    • attributes → object · extra HTML attributes to be added to the text area element.
    • auto_tab → boolean · press tab key to indent?
    • auto_close → object · list of character as the object key that will be auto–completed by its value right after the text area caret.
    • languages → object · list of language string used by the text editor GUI.
    • classes → object · list of HTML class names used by the text editor GUI.
  • Inherits TE.ui.HTML()
    • auto_encode_html → boolean · automatically encode HTML special characters inside code tags?
    • auto_p → boolean · automatically insert paragraph tags?
    • states → object
      • tr → array · minimum and maximum table rows value for table tool.
      • td → array · minimum and maximum table columns value for table tool.
      • abbr → object · list of default abbreviation for abbreviation tool.
    • advance_a → boolean · detect external URL and automatically adds rel="nofollow" and target="_blank" attribute to the link markup?
    • advance_img → boolean · insert image with a <figure> element if title field is defined?
    • advance_table → boolean · include <thead>, <tbody> and <tfoot> markup?
    • formats → object · list of default HTML formats to be outputted.
