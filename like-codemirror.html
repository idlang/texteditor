How it works?

Three layers. The third layer is used to put our textarea with transparent background and transparent text color. So from here, you are missing the caret visibility. The second layer handle it. It is just a <pre> element with transparent background and transparent text, inside it we put a fake caret in the proper position with a help from the library. The last, the first layer is just for placing the pretty printed code. You can use any JavaScript syntax highlighter that available in the web. Here I use Google Code Prettify

Conclusion

You are writing in a transparent textarea. Every time you type something in it, it will update the HTML content of layer 2 and layer 1. That’s it.
