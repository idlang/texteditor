Keyboard Shortcuts

  • ⌘+b → Bold
  • ⌘+g → Image
  • ⌘+h → Header
  • ⌘+i → Italic
  • ⌘+k → Code
  • ⌘+l → Link
  • ⌘+q → Quote
  • ⌘+r → Horizontal Rule
  • ⌘+t → Table
  • ⌘+u → Underline
  • ⌘+y → Redo
  • ⌘+z → Undo
  • ⌘++ → List
  • ⌘+- → Bullet
  • ⌘+✘ → Strike
  • ⌘+↓ → Subscript
  • ⌘+↑ → Superscript
  • ⌘+↵ → Paragraph
  • ⌘+⇧+? → Abbreviation
  • ⇧+↵ → Line Break
  • ✘ → Clear Format → Delete
  • ⌘+⌥+v → Preview
  • ⇥ → Indent & Depends
  • ⇧+⇥ → Outdent
  • ↵ → Depends
  • ⌫ → Depends
