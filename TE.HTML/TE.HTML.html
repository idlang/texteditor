Keyboard Shortcuts

  • ⌘+b → Bold
  • ⌘+g → Image
  • ⌘+h → Header
  • ⌘+i → Italic
  • ⌘+k → Code
  • ⌘+l → Link
  • ⌘+q → Quote
  • ⌘+r → Horizontal Rule
  • ⌘+u → Underline
  • ⌘+y → Redo
  • ⌘+z → Undo
  • ⌘++ → List
  • ⌘+- → Bullet
  • ⌘+✘ → Strike
  • ⌘+↓ → Subscript
  • ⌘+↑ → Superscript
  • ⌘+↵ → Paragraph
  • ⇧+↵ → Line Break
  • ✘ → Clear Format → Delete
  • ↵ → Depends

Depends → Special Cases
