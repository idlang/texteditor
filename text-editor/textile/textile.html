Textile Text Editor

Tips

  1. To create reference–style links, select some text, click the link button, type a link ID in the prompt value, then submit.
  2. To create automatic link syntax, select some text started with http:// or https://, then click the link button.
  3. Default abbreviations can be defined in the config.states.abbr.
  4. Default reference links can be defined in the config.states.a (see the source code of this page).
