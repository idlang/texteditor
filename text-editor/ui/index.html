Text Editor › Core › UI

This is a module to create the GUI appearance as you can see in the text editor below.

Start

Include ui.min.css in the <head> section, include text-editor.min.js and ui.min.js next to the text area:

<!DOCTYPE html>
<html dir="ltr">
  <head>
    <meta charset="utf-8">
    <title>Test</title>
    <link href="text-editor/ui/ui.min.css" rel="stylesheet">
  </head>
  <body>
    <p><textarea></textarea></p>
    <script src="text-editor.min.js"></script>
    <script src="text-editor/ui/ui.min.js"></script>
    <script>
    var editor = TE.ui(document.querySelector('textarea'));
    </script>
  </body>
</html>

Configure

Instance

Initialize a brand new text editor:

var editor = TE.ui(document.querySelector('#editor'));

TE.ui() contains some default options which you can override it through the second parameter:

var editor = TE.ui(document.querySelector('#editor'), {
    // options goes here…
});

Configuration

The available configuration data are:

  • tab → string · Set custom indent character.
  • dir → string · Editor language direction (currently this option only set a HTML dir attribute to the text editor container).
  • resize → boolean · Enable/disable text area resize handler.
  • keys → boolean · Enable/disable keyboard shortcut feature.
  • tools → string · List of tools to be included to the tools bar, each of them separated by a space.
  • attributes → object · Extra HTML attributes to be added to the text area element.
  • auto_tab → boolean · Press Tab key to indent?
  • auto_close → object · List of character as the object key that will be auto–closed by its value right after the text area caret.
  • languages → object · List of words/phrases used by the text editor GUI.
  • classes → object · List of HTML classes used by the text editor GUI.

Custom themes can be included just after the ui.min.css:

<link href="text-editor/ui/ui.min.css" rel="stylesheet">
<link href="text-editor/ui/ui/ui.white.min.css" rel="stylesheet">

Components

Set callback function on the optional fn parameter.

Tool test

editor.ui.tool('emoticon', {
    i: 'smile-o',
    title: 'Emoticon',
    click: function(e, $, id) {
        // `e` refers to the `HTMLEvents` instance
        // `$` refers to the `TE` instance
        // `id` refers to `emoticon` where possible
        return $.insertBefore(' :) '), false;
    }
});
  • title → The tool title.
  • i → Icon identifier or key alias.
  • click → Function that will be executed on click event.
  • text → Use text instead of icon?
  • active → true by default, it means that the tool is active and can be used.
  • attributes → Extra HTML attributes for the tool markup.
  • target → HTML element used to trigger the tool item.

Tools

Overlay test

editor.ui.overlay(true, fn); // `true` to remove overlay by click on it

Panel test

Modal test

editor.ui.modal('my-modal', {
    header: 'Modal Header',
    body: 'Modal body.',
    footer: 'Modal footer.'

    button_2 = TE._.el('button', 'Cancel');

    header: 'Modal Header',
    body: 'Modal body.',
    footer: [button_1, button_2]

    header: 'Modal Header',
    body: 'Modal body.',

        'OK': function(e, $) {
            alert('ok');
        },
        'Cancel': function(e, $) {
            alert('cancel');
        }
    }
}, fn);

        'cancel': function(e, $) { … }
    }
}, fn);

Modal › Prompt

  • title → Prompt title.
  • placeholder → Prompt input placeholder (or default value if required is false).
  • required → Set to true to force user to change the input value other than placeholder and empty value to make it possible to submit the prompt.
  • type → Prompt input type (optional, default is 0):
    • 0 → <input name="data" type="text"> test
    • 1 → <textarea name="data"></textarea> test
    • 2 → <select name="data"> … </select> test
editor.ui.prompt(title, placeholder, required, accept, type);
editor.ui.prompt('Prompt Title', 'Prompt value…', true, function(e, $, value, placeholder) {
    console.log(arguments);
}, 0);

Utilities

Added more helper functions to the TE._ object:

Time

Get time data as year, month, date, hour, minute, second and millisecond:

console.log(TE._.time()); // return `["2016", "11", "26", "17", "25", "23", "258"]`

Replace

Multiple replace function using array:

var from = ['&', '<', '>'];
var to = ['&amp;', '&lt;', '&gt;'];
var encoded = TE._.replace(input, from, to);

The string array data is always global. Therefore, you can also use regular expression as the from array data:

TE._.replace(input, [/\s+/g, '&'], ["", '&amp;']);

Format

Replace pattern in a string with custom text. A pattern is numbers prefixed by a %. Each number will be replaced by data item from an array based on its matching index (1–based index):

var s = TE._.format('Lorem %1 ipsum dolor %2 sit amet.', ['apple', 'orange']);

DOM

Create a new element (the node):

var node = TE._.el('div'); // return `<div></div>`
var node = TE._.el('div', 'test'); // return `<div>test</div>`
var node = TE._.el('div', 'test', {
    'id': 'foo',
    'class': 'bar'
}); // return `<div id="foo" class="bar">test</div>`

Special attributes are classes, data and css:

TE._.el('div', "", {
    'classes': ['foo', 'bar', 'baz']
}); // return `<div class="foo bar baz"></div>`
TE._.el('div', "", {
    'data': {
        'foo': 0,
        'bar': 1,
        'baz': 2
    }
}); // return `<div data-foo="0" data-bar="1" data-baz="2"></div>`
TE._.el('div', "", {
    'css': {
        'background': '#fff',
        'color': '#000'
    }
}); // return `<div style="background: #fff; color: #000;"></div>`

DOM Helpers

Helper functions for DOM manipulation. Asumming node and container is a return value of TE._.el.

Set

Append node to container:

TE._.dom.set(container, node);

Reset

Remove node from container:

TE._.dom.reset(container, node);

Get

Get node from container context:

TE._.dom.get(node, container); // check if `node` is a child of `container`
TE._.dom.get('#foo', container); // is equal to `[container.getElementById('foo')]`
TE._.dom.get('.foo', container); // is equal to `container.getElementsByClassName('foo')`
TE._.dom.get('#foo .bar', container); // is equal to `container.querySelectorAll('#foo .bar')`

If context is not defined, it will be referenced to the document.

Is

Check if node is an <i> tag:

if (TE._.dom.is(node, 'i')) { … }

ID

Setting a unique ID to node if it does not have an id attribute:

TE._.dom.id(node);

Copy

Clone node:

var clone = TE._.dom.copy(node);

Class

Add/remove/toggle and check HTML classes:

TE._.dom.classes.set(node, 'active');
TE._.dom.classes.reset(node, 'active');
TE._.dom.classes.toggle(node, 'active');
TE._.dom.classes.get(node, 'active'); // check if `node` contains `active` class

Attribute

Add/remove and check HTML attributes:

TE._.dom.attributes.set(node, 'title', 'Test title.'); // set `title` attribute with value of `Test title.`
TE._.dom.attributes.set(node, {
    'title': 'Test title.',
    'rel': 'nofollow',
    'target': '_blank'
});

Set an attribute with value of null will removes that attribute.

TE._.dom.attributes.reset(node, 'title'); // remove `title` attribute from `node`
TE._.dom.attributes.reset(node, ['title', 'rel', 'target']); // remove `title`, `rel` and `target` attributes from `node`
TE._.dom.attributes.get(node, 'title'); // get `title` attribute value from `node`
TE._.dom.attributes.get(node, ['title', 'rel', 'target']); // get `title`, `rel` and `target` attributes value from `node`

Data

Add/remove and check HTML data-* attributes:

TE._.dom.data.set(node, 'user', 'ta.tau.taufik'); // set `data-user` attribute with value of `ta.tau.taufik`
TE._.dom.data.set(node, {
    'user': 'ta.tau.taufik',
    'pass': 'foo bar baz'
});

Setting an attribute with value of null will removes that attribute.

TE._.dom.data.reset(node, 'user'); // remove `data-user` attribute from `node`
TE._.dom.data.reset(node, ['user', 'pass']); // remove `data-user` and `data-pass` attributes from `node`
TE._.dom.data.get(node, 'user'); // get `data-user` attribute value from `node`
TE._.dom.data.get(node, ['user', 'pass']); // get `data-user` and `data-pass` attributes value from `node`

Offset

Get node offset relative to its parent:

console.log(TE._.dom.offset(node)); // return `{"l": 40, "t": 20}`

Coordinate

Get current mouse pointer offset relative to node:

node.addEventListener("mousemove", function(e) {
    console.log(TE._.dom.pointer(this, e)); // return `{"x": 4, "y": 8}`
}, false);

Size

Get outer width and outer height of node:

console.log(TE._.dom.size(node)); // return `{"w": 100, "h": 50}`

Content

Currently, this is just an alias for node.innerHTML = 'foo':

TE._.dom.content.set(node, 'foo');

Currently, this is just an alias for node.innerHTML = "":

TE._.dom.content.reset(node);

Currently, this is just an alias for console.log(node.innerHTML):

console.log(TE._.dom.content.get(node));

Parent

Get parent element of node:

var parent = TE._.dom.parent(node);

Children

Get children elements of node:

var children = TE._.dom.children(node);

Next

Get next element from node:

var next = TE._.dom.next(node);

Previous

Get previous element from node:

var next = TE._.dom.previous(node);

Index

Get node position from its parent:

var i = TE._.dom.index(node);

Before

Put node before container:

TE._.dom.before(container, node);

After

Put node after container:

TE._.dom.after(container, node);

Prepend

Prepend node to container:

TE._.dom.prepend(container, node);

Append

Append node to container:

TE._.dom.append(container, node);

Events

Set

Set a click event to node:

function do_click(e) {
    console.log(e.target);
}

TE._.event.set("click", node, do_click);

Each event set will also set a new internal hook data named as on:%s (on:click in this case).

Reset

Remove a click event with do_click as the callback function from node:

TE._.event.reset("click", node, do_click);

Fire

Programmatically trigger click events attached to node:

TE._.event.fire("click", node, [ … ]); // `[ … ]` is for function parameter data

Hooks

Set

Set a new focus.textarea hook:

editor._.hook.set("focus.textarea", function() {
    console.log('focus');
});

Set a new focus.textarea hook with ID of foo:

editor._.hook.set("focus.textarea", function() {
    console.log('focus');
}, 'foo');

Reset

Remove all focus.textarea hooks:

editor._.hook.reset("focus.textarea");

Remove a focus.textarea hook with ID of foo:

editor._.hook.reset("focus.textarea", 'foo');

Fire

Trigger all focus.textarea hooks:

editor._.hook.fire("focus.textarea", [ … ]); // `[ … ]` is for function parameter data

Trigger a focus.textarea hook with ID of foo:

editor._.hook.fire("focus.textarea", [ … ], 'foo');

These methods are not static because every GUI instances will have their own hooks storage.
