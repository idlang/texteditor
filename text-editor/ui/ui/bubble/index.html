Text Editor › Core › UI › Bubble Tools

Floating tools bar.

Start

Include the script just after the plugin instances, then set a new configuration data called bubble to true:

<script src="../text-editor.min.js"></script>
<script src="../text-editor/ui.min.js"></script>
<script>
var editor = TE.ui(document.querySelector('textarea'), {
    bubble: true
});
TE.plug('../text-editor/ui/ui/bubble.min.js');
</script>

Configure

Custom Tools

editor.update({
    bubble: 'b i | a img'
});
editor.update({
    bubble: [
        'b i u', // span
        'p hr table' // block
    ]
});
