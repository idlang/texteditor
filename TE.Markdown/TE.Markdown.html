Markdown Text Editor

Keyboard Shortcuts

  • ⌘+b → Bold
  • ⌘+g → Image
  • ⌘+h → Header
  • ⌘+i → Italic
  • ⌘+k → Code
  • ⌘+l → Link
  • ⌘+q → Quote
  • ⌘+r → Horizontal Rule
  • ⌘+y → Redo
  • ⌘+z → Undo
  • ⌘++ → List
  • ⌘+- → Bullet
  • ⌘+✘ → Strike
  • ⌘+↵ → Paragraph
  • ⇧+↵ → Line Break (Hard Break)
  • ⌘+⌥+v → Preview
  • ⇥ → Indent
  • ⇧+⇥ → Outdent
  • ↵ → Depends
  • ⌫ → Depends

Depends → Special Cases

Try pressing the ↵ key in these position, where | represent the text editor caret.

lorem ipsum dolor sit amet|
<p>|</p>
<p class="foo">|</p>
<p>foo|</p>
<p class="foo">foo|</p>
    |
"|"
'|'
(|)
{|}
[|]
<|>

Try pressing the ⌫ key in these position, where | represent the text editor caret.

<p class="foo"></p>|
<p class="foo">|</p>
    |
